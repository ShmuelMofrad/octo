# octo
## Java 8 parent pom
octo is unus.one pom. general settings for all Java 8 unus.one's projects.

How to use it?
--------------
Start out by adding the parent configuration to your pom.

    <parent>
      <groupId>one.unus</groupId>
      <artifactId>octo</artifactId>
	  <version>1.0.0</version>
    </parent>
